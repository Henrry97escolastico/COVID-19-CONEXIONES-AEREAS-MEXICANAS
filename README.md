 <h1>¿Cómo el ciudadano mexicano llegó a contagiarse?</h1
        <p>Las redes vuelos internacionales(RIV) fueron la vía principal en el contagio del Covid-19, eso provocó que en México el virus se instalara con mayor facilidad y rapidez            mediante la red de vuelos nacionales. Estas tuvieron menos tiempo para reaccionar y sirvieron de advertencia al resto.</p>
